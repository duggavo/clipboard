# Clipboard for Go

Provide copying and pasting to the Clipboard for Go.

Build:

```bash
go get github.com/duggavo/clipboard
```

Platforms:

* OSX
* Windows 7 (probably work on other Windows)
* UNIX/Linux

Documentation:

* https://pkg.go.dev/github.com/duggavo/clipboard
